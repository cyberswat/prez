What is a StackScript?

StackScripts provide Linode Platform users with the ability to automate the deployment of custom systems without requiring maintenance on static images. Linodes deployed with a StackScript run the script as part of the first boot process.

StackScripts are like a bash script with an additional syntax that allow you to request variables from a user.
